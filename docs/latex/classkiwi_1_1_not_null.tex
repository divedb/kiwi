\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class kiwi\+::\+Not\+Null$<$ T $>$}
Restricts a pointer or smart pointer to only hold non-\/null values. And it has zero size overhead over T. 

If T is a pointer (i.\+e. T == U\texorpdfstring{$\ast$}{*}) then
\begin{DoxyItemize}
\item allow construction from U\texorpdfstring{$\ast$}{*}
\item disallow construction from nullptr\+\_\+t
\item disallow default construction
\item ensure construction from null U\texorpdfstring{$\ast$}{*} fails
\item allow implicit conversion to U\texorpdfstring{$\ast$}{*}
\end{DoxyItemize}
