\doxysubsection{Detailed Description}
\doxylink{classkiwi_1_1_string_ref}{String\+Ref} -\/ Represent a constant reference to a string, i.\+e. a character array and a length, which need not be null terminated.

This class does not own the string data, it is expected to be used in situations where the character data resides in some other buffer, whose lifetime extends past that of the \doxylink{classkiwi_1_1_string_ref}{String\+Ref}. For this reason, it is not in general safe to store a \doxylink{classkiwi_1_1_string_ref}{String\+Ref}.

If {\ttfamily Separator} is not in the string, then the result is a pair (LHS, RHS) where (\texorpdfstring{$\ast$}{*}this == LHS) and (RHS == "{}"{}).
