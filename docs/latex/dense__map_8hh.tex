Equality comparison for Dense\+Map.

Iterates over elements of LHS confirming that each (key, value) pair in LHS is also in RHS, and that no additional pairs are in RHS. Equivalent to N calls to RHS.\+find and N value comparisons. Amortized complexity is linear, worst case is O(\+N\texorpdfstring{$^\wedge$}{\string^}2) (if every hash collides).
