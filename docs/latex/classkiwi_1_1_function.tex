The noexcept requires some explanation. {\ttfamily is\+\_\+in\+\_\+situ} is true when the decayed type fits within the internal buffer and is noexcept-\/movable. But this ctor might copy, not move. What we need here, if this ctor does a copy, is that this ctor be noexcept when the copy is noexcept. That is not checked in {\ttfamily is\+\_\+in\+\_\+situ}, and shouldn\textquotesingle{}t be, because once the {\ttfamily \doxylink{classkiwi_1_1_function}{Function}} is constructed, the contained object is never copied. This check is for this ctor only, in the case that this ctor does a copy.
