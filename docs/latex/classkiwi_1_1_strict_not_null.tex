\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class kiwi\+::\+Strict\+Not\+Null$<$ T $>$}
Restricts a pointer or smart pointer to only hold non-\/null values.
\begin{DoxyItemize}
\item provides a strict (i.\+e. explicit constructor from T) wrapper of not\+\_\+null
\item to be used for new code that wishes the design to be cleaner and make not\+\_\+null checks intentional, or in old code that would like to make the transition.
\end{DoxyItemize}

To make the transition from not\+\_\+null, incrementally replace not\+\_\+null by strict\+\_\+not\+\_\+null and fix compilation errors.

Expect to
\begin{DoxyItemize}
\item remove all unneeded conversions from raw pointer to not\+\_\+null and back
\item make API clear by specifying not\+\_\+null in parameters where needed
\item remove unnecessary asserts 
\end{DoxyItemize}
