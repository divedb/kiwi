\doxysubsection{Detailed Description}
A base class for data structure classes wishing to make iterators ("{}handles"{}) pointing into themselves fail-\/fast. When building without asserts, this class is empty and does nothing.

\doxylink{classkiwi_1_1_debug_epoch_base}{Debug\+Epoch\+Base} does not by itself track handles pointing into itself. The expectation is that routines touching the handles will poll on is\+Handle\+In\+Sync at appropriate points to assert that the handle they\textquotesingle{}re using is still valid.
