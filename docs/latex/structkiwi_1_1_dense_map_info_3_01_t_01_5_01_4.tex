\subsubsection*{template$<$typename T$>$\newline
struct kiwi\+::\+Dense\+Map\+Info$<$ T \texorpdfstring{$\ast$}{*} $>$}
Provide \doxylink{structkiwi_1_1_dense_map_info}{Dense\+Map\+Info} for all pointers. Come up with sentinel pointer values that are aligned to alignof(\+T) bytes, but try to avoid requiring T to be complete. This allows clients to instantiate \doxylink{classkiwi_1_1_dense_map}{Dense\+Map$<$\+T\texorpdfstring{$\ast$}{*}, ...$>$} with forward declared key types. Assume that no pointer key type requires more than 4096 bytes of alignment.
