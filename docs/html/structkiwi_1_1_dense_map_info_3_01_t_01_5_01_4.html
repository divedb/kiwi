Detailed Description

template<typename T>
struct kiwi::DenseMapInfo< T * >

Provide DenseMapInfo for all pointers. Come up with sentinel pointer values that are aligned to alignof(T) bytes, but try to avoid requiring T to be complete. This allows clients to instantiate DenseMap<T*, ...> with forward declared key types. Assume that no pointer key type requires more than 4096 bytes of alignment.
