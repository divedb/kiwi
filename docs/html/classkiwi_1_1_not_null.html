Detailed Description

template<typename T>
class kiwi::NotNull< T >

Restricts a pointer or smart pointer to only hold non-null values. And it has zero size overhead over T.

If T is a pointer (i.e. T == U*) then

  • allow construction from U*
  • disallow construction from nullptr_t
  • disallow default construction
  • ensure construction from null U* fails
  • allow implicit conversion to U*
Template Parameters
TMust be type of pointer or comparable with pointer.
