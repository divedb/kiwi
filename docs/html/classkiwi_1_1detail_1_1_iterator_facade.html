Detailed Description

template<typename D, typename V, typename Tag>
class kiwi::detail::IteratorFacade< D, V, Tag >

Currently this only supports forward and bidirectional iteration. The derived class must must have definitions for these methods:

void Increment();
void Decrement(); // optional, to be used with bidirectional
reference Dereference() const;
bool IsEqual([appropriate iterator type] const& rhs) const;
kiwi::iterator

These names are consistent with those used by the Boost iterator facade / adaptor classes to ease migration classes in this file.

Template Parameters
DThe deriving class (CRTP).
VThe value type.
TagThe iterator category, one of:
  • std::forward_iterator_tag
  • std::bidirectional_iterator_tag
